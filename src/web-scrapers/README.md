# Information

This directory is full of the web scrapers for the model. This needs to be collected before the model can be ran.

There is no option implemented currently to run the web scrapers individually.

# Instructions

- Make sure you have compatible versions of Google Chrome and chromedriver.exe installed
- Install the dependencies

  ```pip install -r requirements.txt```

- Run GetData.py to run all the web scrapers
- Collected data will be saved in the src/Data directory
